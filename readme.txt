Persiapan:
    Display masih dioptimalkan untuk pengguna resolusi 1366x768, taskbar autohide (windows), dan browser tanpa sidebar jadi perlu disetting
        - klik kanan background desktop>graphic settings>display ke 1366x768
        - taskbar setting autohide
        - browser chrome/firefox/edge/safari setting tanpa sidebar
    
    requirements ada 2 (requirements.txt dan r.txt), menggunakan r.txt instalasi modul virtual environment lebih sering berjalan normal daripada requirements.txt
        - buat virtual environment python==3.10 (optional recomended)
        - pilih salah satu, pip install -r r.txt (recomended) atau pip install -r requirements.txt


Menjalankan:
- pada kondisi environment yang sudah cocok ketik python app.py
- akan muncul link localhost bisa dicopas ke web untuk membuka app
- app seharusnya sudah berjalan normal dan bisa diuji
